# Quarto Pressmark

_Pressmark_ - an elegant, typographic theme for Quarto websites. Inspired by newspapers like [The Atlantic](https://www.theatlantic.com/) and the [Financial Times](https://www.ft.com/) as well as [Edward Tufte's design principles](https://edwardtufte.github.io/tufte-css/). Made for sites centered around text, that do not want to compromise on good design. Feauturing carefully selected fonts, well thought-trough default styling, and a bunch of quirks.

> **Pressmark** _noun_: A notation or figure in the margin of a printed sheet indicating the press on which it was printed. 
>
> *\- The American Heritage® Dictionary of the English Language, 5th Edition.*
